## <a name="clean-up-resources"></a>리소스 정리

데이터 연결을 삭제 하려면 다음 명령을 사용 합니다.

```csharp
kustoManagementClient.DataConnections.Delete(resourceGroupName, clusterName, databaseName, dataConnectionName);
```
