# <a name="exp"></a>exp()

X의 밑이 e 인 지 수 함수 이며 e는 거듭제곱 x: e ^ x입니다.  

## <a name="syntax"></a>구문

`exp(`*.x*`)`

## <a name="arguments"></a>인수

* *x*: 지 수 값의 실수입니다.

## <a name="returns"></a>반환

* X의 지 수 값입니다.
* 자연 (밑수-e) [로그는 log ()](log-function.md)를 참조 하세요.
* 밑이 2 인 지 수 함수 및 밑이 10 인 경우 [exp2 ()](exp2-function.md), [exp10 ()](exp10-function.md) 을 참조 하세요.
