# <a name="time-series-analysis"></a>Time Series Analysis 

Kusto 쿼리 언어는 기본 데이터 형식으로 시리즈에서 지원을 제공합니다.
연산자 메이크 계열은 데이터를 계열 데이터 유형으로 변환하고 이 데이터 형식의 고급 처리를 위해 함수 제품군이 제공됩니다. 이 기능에는 데이터 정리, 기계 학습 모델링 및 이상값 감지가 포함됩니다.
