# <a name="atan"></a>atan()

탄젠트가 지정 된 숫자 (의 역 연산) 인 각도를 반환 [`tan()`](tanfunction.md) 합니다.

## <a name="syntax"></a>구문

`atan(`*.x*`)`

## <a name="arguments"></a>인수

* *x*: 실수입니다.

## <a name="returns"></a>반환

* 의 아크 탄젠트 값입니다. `x`
