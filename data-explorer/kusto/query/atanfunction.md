# <a name="atan"></a>atan()

접선이 지정된 숫자(역작업)인 각도를 [`tan()`](tanfunction.md)반환합니다.

**구문**

`atan(`*Ⅹ*`)`

**인수**

* *x*: 실제 숫자입니다.

**반환**

* 의 호 접선값`x`
