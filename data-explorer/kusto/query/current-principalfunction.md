# <a name="current_principal"></a>current_principal()

::: zone pivot="azuredataexplorer"

쿼리를 실행 하는 현재 사용자 이름을 반환 합니다.

## <a name="syntax"></a>Syntax

`current_principal()`

## <a name="returns"></a>반환

현재 보안 주체의 FQN (정규화 된 이름) `string` 입니다.  
문자열 형식은 다음과 같습니다.  
*PrinciplaType* `=` *Principalid* `;` *TenantId*
