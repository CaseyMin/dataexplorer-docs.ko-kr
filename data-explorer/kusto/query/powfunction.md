# <a name="pow"></a>pow()

전원을 들어 올린 결과 반환

**구문**

`pow(`*기본* `,` *지수*`)`

**인수**

* *기본*: 기본 값입니다.
* *지수*: 지수 값입니다.

**반환**

전원 지수로 상승된 기본 을 반환합니다.
