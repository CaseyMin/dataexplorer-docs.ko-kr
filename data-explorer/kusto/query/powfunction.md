# <a name="pow"></a>pow()

거듭제곱 한 결과를 반환 합니다.

## <a name="syntax"></a>Syntax

`pow(`*기본* `,` *지* 수`)`

## <a name="arguments"></a>인수

* *base*: 기준 값입니다.
* *지수가*: 지 수 값입니다.

## <a name="returns"></a>반환

거듭제곱 거듭제곱의 밑수를 반환 합니다. base ^ 지수가 반환 됩니다.
