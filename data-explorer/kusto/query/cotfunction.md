# <a name="cot"></a>cot()

지정 된 각도의 삼각 코탄젠트를 라디안으로 계산 합니다.

## <a name="syntax"></a>Syntax

`cot(`*.x*`)`

## <a name="arguments"></a>인수

* *x*: 실수입니다.

## <a name="returns"></a>반환

* 에 대 한 코탄젠트 함수 값`x`
