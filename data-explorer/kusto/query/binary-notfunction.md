# <a name="binary_not"></a>binary_not()

입력 값의 비트 부정을 반환 합니다.

```kusto
binary_not(x)
```

## <a name="syntax"></a>구문

`binary_not(`*num1*`)`

## <a name="arguments"></a>인수

* *num1*: numeric 

## <a name="returns"></a>반환

숫자에 대 한 논리적 NOT 연산 (num1)을 반환 합니다.
