# <a name="loggamma"></a>loggamma()

[감마 함수의](https://en.wikipedia.org/wiki/Gamma_function) 절대 값 로그 계산

**구문**

`loggamma(`*Ⅹ*`)`

**인수**

* *x*: 감마 함수에 대한 매개 변수

**반환**

* x의 감마 함수의 절대 값의 자연 엽을 반환합니다.
* 감마 함수를 계산하는 경우 [감마()](gammafunction.md)을 참조하십시오.
