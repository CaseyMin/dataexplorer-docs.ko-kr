# <a name="loggamma"></a>loggamma()

[감마 함수의](https://en.wikipedia.org/wiki/Gamma_function) 절대값 로그를 계산 합니다.

## <a name="syntax"></a>Syntax

`loggamma(`*.x*`)`

## <a name="arguments"></a>인수

* *x*: 감마 함수에 대 한 매개 변수

## <a name="returns"></a>반환

* X의 감마 함수 절대 값의 자연 로그를 반환 합니다.
* 계산 감마 함수는 [감마 ()](gammafunction.md)를 참조 하세요.
