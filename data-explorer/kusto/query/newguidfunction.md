# <a name="new_guid"></a>new_guid()

임의의 GUID (Globally Unique Identifier)를 반환 합니다.

```kusto
new_guid()
```

## <a name="syntax"></a>Syntax

* `new_guid()`-형식의 새 값을 생성 `guid` 합니다.
