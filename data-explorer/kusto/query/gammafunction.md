# <a name="gamma"></a>gamma()

[감마 함수](https://en.wikipedia.org/wiki/Gamma_function) 를 계산 합니다.

## <a name="syntax"></a>구문

`gamma(`*.x*`)`

## <a name="arguments"></a>인수

* *x*: 감마 함수에 대 한 매개 변수

## <a name="returns"></a>반환

* X의 감마 함수입니다.
* 로그 감마 함수를 계산 하려면 [loggamma ()](loggammafunction.md)를 참조 하세요.
