# <a name="gamma"></a>gamma()

[감마 함수](https://en.wikipedia.org/wiki/Gamma_function) 계산

**구문**

`gamma(`*Ⅹ*`)`

**인수**

* *x*: 감마 함수에 대한 매개 변수

**반환**

* x의 감마 함수.
* 로그 감마 함수를 계산하는 경우 [loggamma()](loggammafunction.md)을 참조하십시오.
