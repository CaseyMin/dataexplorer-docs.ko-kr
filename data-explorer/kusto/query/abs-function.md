# <a name="abs"></a>abs()

입력의 절대값을 계산합니다.  

**구문**

`abs(`*Ⅹ*`)`

**인수**

* *x*: 정수 또는 실수 또는 시간 범위 값입니다.

**반환**

* x의 절대 값입니다.
