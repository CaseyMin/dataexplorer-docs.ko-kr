# <a name="abs"></a>abs()

입력의 절대값을 계산 합니다.  

## <a name="syntax"></a>구문

`abs(`*.x*`)`

## <a name="arguments"></a>인수

* *x*: 정수 또는 실수 또는 timespan 값입니다.

## <a name="returns"></a>반환

* X의 절대값입니다.
