탄젠트 함수를 반환 합니다.

## <a name="syntax"></a>구문

`tan(`*.x*`)`

## <a name="arguments"></a>인수

* *x*: 실수입니다.

## <a name="returns"></a>반환

* 의 결과입니다.`tan(x)`
