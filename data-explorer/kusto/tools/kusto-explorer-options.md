# <a name="kusto-explorer-options"></a>쿠스토 익스플로러 옵션

다음 표에서는 **도구** > **옵션** 대화 상자에서 Kusto.Explorer의 동작을 사용자 지정하는 옵션에 대해 설명합니다.

![** 도구 ** > **옵션 **](\images\Kusto-Tools-Explorer-Options\tools-options.png)

## <a name="general-settings"></a>General Settings(일반 설정)

| 옵션 | Description |
|---------|--------------|
| 공구 모드 | 베타, 알파 및 실험 응용 프로그램 기능을 활성화합니다. 기본값은 none입니다.|
| 확장된 접근성 | 이 옵션을 사용하면 응용 프로그램이 내게 필요한 옵션 도구에서 사용하는 내게 필요한 옵션 이벤트를 보냅니다. 활성화된 경우 성능에 영향을 줄 수 있습니다. 기본값은 사용할 수 없습니다. |
| 원격 분석 데이터 전송 허용 | 활성화되면 응용 프로그램은 오류가 발생하거나 응용 프로그램이 충돌할 때 원격 분석 데이터를 보냅니다. |
| 환영 메시지 | 활성화하면 응용 프로그램이 시작 시 시작 메시지를 표시합니다. 기본값이 활성화되어 있습니다.|
| 테마 표시 | 응용 프로그램 UI 색상 구성표: 밝거나 어두운.|
  
## <a name="query-editor"></a>쿼리 편집기

| 옵션 | Description |
|---------|--------------|
| 쿼리 자동 저장 | 활성화하면 응용 프로그램이 열려 있는 문서를 자동으로 저장합니다. 이 설정을 변경하려면 응용 프로그램을 다시 시작해야 합니다. 기본적으로 사용하도록 설정되어 있습니다.|
| 변경 내용 추적 작업 | 이 옵션을 사용하면 응용 프로그램은 다른 응용 프로그램에서 디스크의 쿼리 스크립트에 대한 변경 내용을 추적합니다. 쿼리 스크립트가 외부에서 변경되면 알림을 받고 다시 로드하라는 메시지가 표시됩니다. 이 설정을 변경하려면 응용 프로그램을 다시 시작해야 합니다. 기본적으로 사용하도록 설정되어 있습니다.|
| 세션 편집 사용 | 활성화된 경우 각 응용 프로그램 프로세스는 고유한 문서 집합을 소유합니다. 기본적으로 사용하지 않도록 설정됩니다.|
| 글꼴 크기 | 쿼리 편집기에서 사용되는 글꼴 크기입니다.|
| 명령 배경 선택 | 현재 선택한 명령을 강조 표시하는 데 사용되는 배경 색입니다.|
| 탭을 공백으로 바꾸기 | 활성화하면 탭이 자동으로 공백으로 바뀝니다.|
| 기능 매개 변수 주입 을 사용하지 않도록 설정 | 활성화하면 클립보드의 함수 매개변수 삽입이 비활성화됩니다.|
| 쿼리 매개 변수 삽입 을 사용하지 않도록 설정 | 활성화하면 쿼리 매개 변수 삽입이 비활성화됩니다.|
| F5를 제외한 쿼리 실행 트리거 사용 안 함 | 활성화하면 F5 키만 쿼리가 실행되도록 트리거됩니다.|
| 응용 프로그램 내부 도움말 표시 | 활성화되면 도움말 항목이 응용 프로그램 내부에 표시됩니다. 비활성화된 도움말 항목이 브라우저 내에서 열리면|
| 쿼리 매개 변수 길이 제한 | 쿼리 매개 변수로 사용할 수 있는 문자열의 최대 길이입니다. 이 값을 128K 이상으로 설정하면 성능 문제가 발생할 수 있습니다. 기본값은 64K입니다.|

## <a name="intellisense"></a>IntelliSense

| 옵션 | Description |
|---------|--------------|
| 인텔리센스 버전 | 사용 된 IntelliSense 엔진의 버전. *V1은* 고전적인 엔진입니다. *V2는* 현대적인 엔진입니다. 기본값은 *V2*입니다. |
| 인텔리센스: 제어 명령 | 제어 명령에 대한 IntelliSense의 버전입니다. *V1은* 고전적인 엔진입니다. *V2는* 현대적인 엔진입니다. 기본값은 *V2*입니다. | 
| IntelliSense | IntelliSense를 활성화하거나 사용하지 않도록 설정합니다. 기본값이 활성화되어 있습니다.|
| 구문 강조 표시 | 구문 강조 표시를 사용하거나 사용하지 않도록 설정합니다. 기본값이 활성화되어 있습니다.|
| 도구 팁 | 마우스가 선택한 쿼리 영역 위로 마우스를 가져간 경우 나타나는 도구 팁을 활성화하거나 사용하지 않도록 설정합니다. 기본값이 활성화되어 있습니다.|

## <a name="formatter"></a>포맷터

| 옵션 | Description |
|---------|--------------|
| 버전 | 현재 쿼리에 예쁘게 표시 쿼리 도구가 적용될 때 사용되는 포터의 버전입니다. *V1은* 고전적인 포터입니다. *V2는* 현대 포터입니다. 기본값은 *V2*입니다.|
| 들여쓰기 | 들여쓰기된 항목의 공백 수입니다.|
| 기능 중괄호 | 함수 중괄호의 배치입니다. *새* 줄에서 세로 배치가 열리고 곱슬 교정기를 닫습니다. *가로는* 열린 중괄호를 원래 줄에 두고 닫기 중괄호를 새 줄에 배치합니다. *아무도* 모든 중괄호를 있는 것처럼 남기지 않습니다.|
| 파이프 연산자 | 테이블 형식 쿼리 연산자 사이에 있는 파이프(막대) 문자의 배치입니다. *새 줄은* 모든 파이프 문자를 새 줄에 배치합니다. *쿼리가* 이미 두 줄 이상에 걸쳐 있는 경우 스마트는 모든 파이프 문자를 새 줄에 배치합니다. *없음은* 모든 파이프 문자를 있는 것처럼 둡니다.|
| 세미콜론 | 쿼리 문을 종료하는 세미콜론의 배치입니다. *New Line은* 세미콜론을 들여쓰기된 새 줄에 배치합니다. *스마트는* 문 자체가 두 개 이상의 줄에 걸쳐 있는 경우 세미콜론을 새 줄에 배치합니다.  *아무도* 세미콜론을 있는 것처럼 남기지 않습니다.
| 누락된 구문 삽입 | 쉼표 및 세미콜론과 같은 누락된 문장 부호를 추가하여 오류 메시지가 표시됩니다.|

## <a name="results-viewer"></a>결과 뷰어

| 옵션 | Description |
|---------|--------------|
| 글꼴 크기 | 쿼리 결과가 표시되는 데이터 테이블 표에서 사용되는 글꼴 크기입니다.|
| 자세한 색 구성표 | 자동 감지된 자세한 수준에서 행 서식 베이스에 대한 색 구성표를 선택합니다.|
| 빈 열 숨기기 | 활성화하면 데이터를 표시하는 뷰의 빈 열이 숨김됩니다.  기본값은 사용할 수 없습니다.|
| 단일 값 열 축소| 활성화하면 데이터를 표시하는 뷰에서 단일 값 열을 자동으로 축소합니다. 비어 없는 단일 값이 있는 열은 그룹으로 표시됩니다. 기본값은 사용할 수 없습니다.|
| 날짜 시간 열별로 결과 자동 정렬 | 활성화하면 행이 datetime 열별로 자동으로 정렬됩니다. 기본값이 활성화되어 있습니다.|
| 열 가시 범위 | 열에 표시할 문자의 최대 양입니다. 기본값은 2048입니다.|
| JSON을 텍스트로 시각화 | 이 옵션을 사용하면 JSON 값이 텍스트로 시각화됩니다. 기본값은 사용할 수 없습니다.|
| 최대 텍스트 세부 정보 크기 | 셀을 두 번 클릭할 때 상세 정보 패널에 표시되는 최대 문자 수입니다. 기본값은 32KB입니다.|

## <a name="connections"></a>Connections

| 옵션 | Description |
|---------|--------------|
| 테이블 열을 사전순으로 정렬 | 활성화하면 연결 패널의 테이블 노드 아래에 나타나는 열이 사전순으로 정렬됩니다.|
| 쿼리 서버 시간 시간 | 쿼리 실행을 위한 서버 시간 시간 입니다.|
| 연결 잠금에 대한 경고 | 활성화되면 연결 스위치를 시도할 때마다 응용 프로그램이 연결 잠금에 대해 경고합니다. 기본값이 활성화되어 있습니다.|
| 지연 스키마 탐색 | 이 옵션을 사용하면 데이터베이스 노드가 확장될 때만 연결 패널이 데이터베이스 스키마를 가져오고 표시합니다.|
| 숨겨진 시스템 개체 표시 | 활성화하면 사용자에게 적절한 권한이 있는 경우 숨겨진 시스템 개체가 표시됩니다.|
| 약한 일관성 쿼리 | 이 옵션을 사용하면 쿼리에 약한 일관성이 사용됩니다.|
| 쿠스토 파서 버전 | 쿼리를 실행하는 데 사용되는 파서 버전입니다. *V1은* 고전적인 파서입니다. *V2는* 현대 파서입니다. 기본값은 *V1*입니다.|
