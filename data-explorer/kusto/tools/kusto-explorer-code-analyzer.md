# <a name="kusto-explorer-code-analyzer"></a>Kusto 탐색기 코드 분석기

Kusto 탐색기는 현재 쿼리를 분석 하 고 적용 가능한 개선 권장 사항 집합을 출력 하는 코드 분석기 유틸리티를 제공 합니다. 

`Ctrl` + `F6` 를 사용 하 여 활성 쿼리에서 분석기를 실행 합니다.
